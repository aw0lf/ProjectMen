### Instalacja

Clone projekt

```
git clone https://github.com/aw0lf/ProjectMen
```
Przejdz do projekt, do katalogu backend/mysite
```
cd /backend/mysite
 ```
 Uruchom virualenv
 Zaintaluj
 ```
 pip install -r requirements.txt 
 ```
 Uruchom server
 ```
 python manage.py runserver
 ```
 Przejdz do katalogu frontend
 ```
 cd ../../frontend
 ```
 Zainstaluj 
 ```
 npn install
 ```
 Uruchom server
 ```

 ```
